# Cardspark UI

React components and design tokens for trading card collection interfaces.

## Workspace

- `packages/ui`: React component package
- `packages/tokens`: internal token source re-exported through `@cardspark/ui/tokens.css`
- `docs/theming.md`: theme authoring notes

## Commands

```bash
npm install
npm run build
npm run typecheck
```

The published package is `@cardspark/ui`.
